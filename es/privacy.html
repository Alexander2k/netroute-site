Política de privacidad

Última actualización: 30 de marzo de 2026

Descripción general

NetRoute Pro es una extensión de navegador que descubre dominios en páginas web, los resuelve a direcciones IP y genera comandos de enrutamiento. Tu privacidad es importante para nosotros. Esta extensión está diseñada para funcionar completamente de forma local en tu navegador.

Recopilación de datos

NetRoute Pro no recopila, almacena, transmite ni comparte datos personales. Específicamente:

  • Sin scripts de análisis o seguimiento
  • Sin cuentas de usuario ni registro
  • Sin cookies de seguimiento
  • Sin datos enviados a nuestros servidores
  • Sin registro del historial de navegación

Solicitudes de red

La extensión realiza las siguientes solicitudes de red, todas esenciales para su funcionalidad principal:

  • Consultas DNS-over-HTTPS a Cloudflare (1.1.1.1), Google (8.8.8.8) y AdGuard (94.140.14.140) — para resolver nombres de dominio a direcciones IP
  • RIPE Stat API (stat.ripe.net) — para obtener información de prefijos BGP para optimización de rutas (opcional)
  • ip-api.com — para buscar información ASN/organización de IPs resueltas (opcional)

Estas solicitudes contienen solo nombres de dominio o direcciones IP descubiertas en la página web actual. No se incluye información personal.

Almacenamiento local

La extensión usa chrome.storage.local y opcionalmente chrome.storage.sync para almacenar:

  • Preferencias del usuario (tema, proveedor DNS, configuración de exportación)
  • Lista negra de dominios
  • Historial de escaneos (últimos 10, almacenados localmente)
  • Caché DNS y BGP (con expiración TTL)

Todos los datos permanecen en tu dispositivo (o se sincronizan a través de tu cuenta de Google si habilitas Chrome Sync). No tenemos acceso a estos datos.

Permisos

  • activeTab — para acceder a la URL de la pestaña actual para escaneo de dominios
  • scripting — para extraer dominios de la página actual
  • storage — para guardar configuración y caché localmente
  • downloads — para exportar archivos de rutas (.bat, .sh, .rsc)
  • webRequest — para capturar solicitudes de red para descubrimiento de dominios
  • host_permissions (todas las URLs) — para realizar consultas DNS-over-HTTPS para resolver dominios

Servicios de terceros

No se comparten datos con terceros para publicidad, marketing o análisis. Los únicos servicios externos contactados son los proveedores DNS y las APIs de información de red listadas arriba, únicamente para la funcionalidad principal de la extensión.

Cambios

Si esta política se actualiza, los cambios se publicarán en esta página con una fecha actualizada.

Contacto
