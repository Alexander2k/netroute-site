Todo lo que necesitas para generar comandos de enrutamiento listos para producción desde cualquier página web.

Sniffer de red

Captura solicitudes en segundo plano — AJAX, CDN, API — en tiempo real mediante Service Worker.

DNS Racing

Consulta Cloudflare, Google, Quad9 simultáneamente via Promise.any. La primera respuesta gana.

Agregación CIDR

Fusiona IPs individuales en subredes optimizadas. Agregación IPv4 bitmask + IPv6 BigInt.

Optimización RIPE BGP

Obtiene prefijos BGP reales de RIPE Stat API para reemplazar /32 individuales con rutas anunciadas.

Consulta ASN

Resolución masiva IP-a-ASN. Ve qué organización posee cada IP: AS13335 (Cloudflare).

Exportación Pro

Preguntas frecuentes

¿Qué plataformas soporta NetRoute Pro?
NetRoute Pro genera comandos de enrutamiento para 8 plataformas: Windows, Linux, macOS, MikroTik, Cisco, Juniper, WireGuard (AllowedIPs) y OpenVPN. Compatible con IPv4 e IPv6.
¿NetRoute Pro es gratis?
Sí, NetRoute Pro es completamente gratis. No requiere registro ni suscripción.
¿Cómo genera NetRoute Pro rutas desde un sitio web?
NetRoute Pro descubre todos los dominios de una página web (incluyendo solicitudes AJAX, CDN y API mediante Service Worker), los resuelve a direcciones IP usando consultas DNS en paralelo a Cloudflare, Google y Quad9, agrega los IPs en subredes CIDR optimizadas y genera comandos de enrutamiento específicos de la plataforma.
¿Se puede usar para VPN split tunneling?
Sí, NetRoute Pro es ideal para enrutamiento selectivo VPN. Genera WireGuard AllowedIPs, directivas route de OpenVPN y comandos de enrutamiento del sistema que te permiten enrutar solo el tráfico de sitios específicos a través de tu VPN.
¿NetRoute Pro recopila datos personales?
No. NetRoute Pro funciona completamente de forma local en tu navegador. No recopila, almacena ni transmite datos personales. Las únicas solicitudes de red son consultas DNS y búsquedas opcionales de RIPE/ASN para la funcionalidad principal de la extensión.
