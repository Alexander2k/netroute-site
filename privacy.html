Privacy Policy

Last updated: March 30, 2026

Overview

NetRoute Pro is a browser extension that discovers domains on webpages, resolves them to IP addresses, and generates routing commands. Your privacy is important to us. This extension is designed to work entirely locally in your browser.

Data Collection

NetRoute Pro does not collect, store, transmit, or share any personal data. Specifically:

  • No analytics or tracking scripts
  • No user accounts or registration
  • No cookies for tracking purposes
  • No data sent to our servers
  • No browsing history recorded

Network Requests

The extension makes the following network requests, all essential to its core functionality:

  • DNS-over-HTTPS queries to Cloudflare (1.1.1.1), Google (8.8.8.8), and Quad9 (9.9.9.9) — to resolve domain names to IP addresses
  • RIPE Stat API (stat.ripe.net) — to fetch BGP prefix information for route optimization (optional, user-enabled)
  • ip-api.com — to look up ASN/organization info for resolved IPs (optional)

These requests contain only domain names or IP addresses discovered on the current webpage. No personal information is included.

Local Storage

The extension uses chrome.storage.local and optionally chrome.storage.sync to store:

  • User preferences (theme, DNS provider, export settings)
  • Domain blacklist
  • Scan history (last 10 scans, stored locally)
  • DNS and BGP cache (with TTL expiration)

All data stays on your device (or syncs via your Google account if you enable Chrome Sync). We have no access to this data.

Permissions

  • activeTab — to access the current tab's URL for domain scanning
  • scripting — to extract domains from the current page
  • storage — to save settings and cache locally
  • downloads — to export route files (.bat, .sh, .rsc)
  • webRequest — to capture network requests for domain discovery
  • host_permissions (all URLs) — to make DNS-over-HTTPS queries to resolve domains

Third-Party Services

No data is shared with third parties for advertising, marketing, or analytics purposes. The only external services contacted are DNS providers and network information APIs listed above, solely for the extension's core functionality.

Changes

If this policy is updated, the changes will be posted on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue on the project's support page.
