Инструкции по маршрутизации

Пошаговые инструкции по применению маршрутов, сгенерированных в NetRoute Pro, на всех популярных роутерах, ОС и VPN-протоколах.

Роутер

Keenetic

Загрузка .bat файла с привязкой к VPN-интерфейсу — одноклик для KeeneticOS.

 Роутер

MikroTik

Импорт маршрутов через .rsc скрипт, address lists и динамический DNS для RouterOS.
Скоро

OpenWrt

Применение маршрутов через UCI, LuCI или командную строку на прошивках OpenWrt.

Скоро

WireGuard

Универсальный подход через AllowedIPs — работает на любом роутере или ОС с WireGuard.

Скоро

OpenVPN

Директивы route и настройка клиентского конфига для split tunneling.

Скоро

Windows

Запуск .bat от администратора или netsh для постоянных маршрутов.

Скоро

Linux

Применение .sh через ip route, systemd-networkd или NetworkManager.

Скоро

macOS

Команда route с sudo или launchd для постоянных статических маршрутов.

Скоро

Cisco

Импорт маршрутов через CLI на IOS/IOS-XE.

Скоро

Juniper

Добавление статических маршрутов через Junos set-команды.
