Создано для сетевых инженеров

Всё, что нужно для генерации production-ready команд маршрутизации с любой веб-страницы.

Сетевой сниффер

Перехватывает фоновые запросы — AJAX, CDN, API — в реальном времени через Service Worker.

DNS Racing

Запрашивает Cloudflare, Google, Quad9 одновременно через Promise.any. Побеждает первый ответ.

CIDR агрегация

Объединяет отдельные IP в оптимизированные подсети. IPv4 bitmask + IPv6 BigInt агрегация.

RIPE BGP оптимизация

Получает реальные BGP-префиксы из RIPE Stat API для замены отдельных /32 на анонсированные маршруты.

ASN поиск

Пакетное определение IP-to-ASN. Узнайте, какой организации принадлежит каждый IP: AS13335 (Cloudflare).

Pro экспорт

Скачивайте как .bat, .sh или .rsc с правильными заголовками и командами проверки.

Массовое сканирование

Вставьте список URL. Сгенерируйте агрегированные маршруты для всех хостов сразу.

Частые вопросы

Какие платформы поддерживает NetRoute Pro?
NetRoute Pro генерирует команды маршрутизации для 8 платформ: Windows, Linux, macOS, MikroTik, Cisco, Juniper, WireGuard (AllowedIPs) и OpenVPN. Поддерживаются IPv4 и IPv6.
NetRoute Pro бесплатный?
Да, NetRoute Pro полностью бесплатен. Не требуется регистрация или подписка.
Как NetRoute Pro генерирует маршруты с сайта?
Расширение находит все домены на веб-странице (включая AJAX, CDN и API запросы через Service Worker), резолвит их в IP-адреса параллельными DNS-запросами к Cloudflare, Google и Quad9, агрегирует IP в оптимизированные CIDR-подсети и генерирует команды маршрутизации для выбранной платформы.
Можно ли использовать для VPN split tunneling?
Да, NetRoute Pro идеально подходит для выборочной маршрутизации VPN. Расширение генерирует WireGuard AllowedIPs, OpenVPN route-директивы и команды ОС, чтобы через VPN шёл трафик только определённых сайтов.
Собирает ли NetRoute Pro персональные данные?
Нет. Расширение работает полностью локально в браузере. Не собирает, не хранит и не передаёт персональные данные. Единственные сетевые запросы — DNS-запросы и опциональные RIPE/ASN-лукапы для работы расширения.
