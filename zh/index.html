// 功能特性

专为网络工程师打造

从任何网页生成生产级路由命令所需的一切。

网络嗅探器

通过Service Worker实时捕获后台请求 — AJAX、CDN、API调用。

DNS竞速

同时查询Cloudflare、Google、Quad9，通过Promise.any，最快响应获胜。

CIDR聚合

将单个IP合并为优化的子网。IPv4位掩码 + IPv6 BigInt聚合。

RIPE BGP优化

从RIPE Stat API获取真实BGP前缀，用已公告路由替换单个/32。

ASN查询

批量IP转ASN解析。查看每个IP的所属组织：AS13335 (Cloudflare)。

专业导出

下载为.bat、.sh或.rsc，包含正确的文件头和验证命令。

批量扫描

粘贴URL列表。一次性为所有主机名生成聚合路由。

域名黑名单

排除分析、追踪器和不需要的域名。通过Chrome Cloud Sync跨设备同步。

// 工作流程

三步生成路由

01

安装并浏览

从Chrome Web Store添加扩展。浏览任何网站并刷新以启动网络嗅探器。

# chrome://extensions
NetRoute Pro enabled
ID: abc123def456
02

配置并扫描

选择目标操作系统、合并掩码、网关和域名过滤器。点击Analyze Website。

# Scan config
OS:      MikroTik
Gateway: 10.0.0.1
Mask:    /24
Filter:  .*cdn.*
03

复制或导出

从Routes标签页获取生成的路由。复制到剪贴板或导出为脚本文件。

# Output: 12 routes generated
/ip route add dst=104.21.32.0/24 \
  gateway=10.0.0.1
/ip route add dst=172.67.0.0/16 \
  gateway=10.0.0.1
