// 功能特性

专为网络工程师打造

从任何网页生成生产级路由命令所需的一切。

网络嗅探器

通过Service Worker实时捕获后台请求 — AJAX、CDN、API调用。

DNS竞速

同时查询Cloudflare、Google、AdGuard，通过Promise.any，最快响应获胜。

CIDR聚合

将单个IP合并为优化的子网。IPv4位掩码 + IPv6 BigInt聚合。

RIPE BGP优化

从RIPE Stat API获取真实BGP前缀，用已公告路由替换单个/32。

ASN查询

批量IP转ASN解析。查看每个IP的所属组织：AS13335 (Cloudflare)。

专业导出

下载为.bat、.sh或.rsc，包含正确的文件头和验证命令。

批量扫描

粘贴URL列表。一次性为所有主机名生成聚合路由。

域名黑名单

排除分析、追踪器和不需要的域名。通过Chrome Cloud Sync跨设备同步。

常见问题

NetRoute Pro 支持哪些平台？
NetRoute Pro 为 8 个平台生成路由命令：Windows、Linux、macOS、MikroTik、Cisco、Juniper、WireGuard（AllowedIPs）和 OpenVPN。支持 IPv4 和 IPv6。
NetRoute Pro 是免费的吗？
是的，NetRoute Pro 完全免费。无需注册或订阅。
NetRoute Pro 如何从网站生成路由？
NetRoute Pro 发现网页上的所有域名（包括通过 Service Worker 的 AJAX、CDN 和 API 请求），使用并行 DNS 查询到 Cloudflare、Google 和 AdGuard 将其解析为 IP 地址，将 IP 聚合为优化的 CIDR 子网，并生成特定于平台的路由命令。
可以用于 VPN 分流吗？
可以，NetRoute Pro 非常适合 VPN 分流。它生成 WireGuard AllowedIPs、OpenVPN 路由指令和操作系统级路由命令，让您仅将特定网站的流量通过 VPN 路由。
NetRoute Pro 会收集个人数据吗？
不会。NetRoute Pro 完全在您的浏览器中本地运行。它不收集、存储或传输任何个人数据。唯一的网络请求是用于扩展核心功能的 DNS 查询和可选的 RIPE/ASN 查询。
