隐私政策

最后更新：2026年3月30日

概述

NetRoute Pro是一款浏览器扩展，它能发现网页上的域名，将其解析为IP地址，并生成路由命令。您的隐私对我们很重要。此扩展完全在您的浏览器中本地运行。

数据收集

NetRoute Pro 不会收集、存储、传输或共享任何个人数据。具体来说：

  • 没有分析或跟踪脚本
  • 没有用户账户或注册
  • 没有用于跟踪的Cookie
  • 没有数据发送到我们的服务器
  • 没有浏览历史记录

网络请求

扩展进行以下网络请求，均为其核心功能所必需：

  • DNS-over-HTTPS查询 发送至Cloudflare (1.1.1.1)、Google (8.8.8.8)和Quad9 (9.9.9.9) — 用于将域名解析为IP地址
  • RIPE Stat API (stat.ripe.net) — 用于获取BGP前缀信息以优化路由（可选）
  • ip-api.com — 用于查询已解析IP的ASN/组织信息（可选）

这些请求仅包含在当前网页上发现的域名或IP地址。不包含任何个人信息。

本地存储

扩展使用chrome.storage.local和可选的chrome.storage.sync来存储：

  • 用户偏好设置（主题、DNS提供商、导出设置）
  • 域名黑名单
  • 扫描历史（最近10次，本地存储）
  • DNS和BGP缓存（带TTL过期）

所有数据都保存在您的设备上（如果您启用Chrome Sync，则通过Google账户同步）。我们无法访问这些数据。

权限

  • activeTab — 访问当前标签页的URL以进行域名扫描
  • scripting — 从当前页面提取域名
  • storage — 本地保存设置和缓存
  • downloads — 导出路由文件（.bat、.sh、.rsc）
  • webRequest — 捕获网络请求以发现域名
  • host_permissions（所有URL） — 执行DNS-over-HTTPS查询以解析域名

第三方服务

不会与第三方共享任何数据用于广告、营销或分析目的。唯一联系的外部服务是上述DNS提供商和网络信息API，仅用于扩展的核心功能。

变更

如果此政策更新，更改将发布在此页面上，并附有更新日期。
